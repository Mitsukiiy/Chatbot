## Getting Started

### Prerequisites

- Python 3.9 or higher

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Mitsukiiy/Chatbot.git
   ```

2. Navigate to the project directory:

   ```bash
   cd chatbot
   ```

3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the chatbot:

   ```bash
   python chatbot.py
   ```
